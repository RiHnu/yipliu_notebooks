# 1. App UI 介绍

## 1.1 Overview Tab


Dashboard 跟踪实验，可视化结果

Report 分享，保存结果

Sweeps: 超参调优

Artifacts: 数据集和模型的版本控制

# 2. init 参数介绍

**project** 项目名称

**name** 本次实验的名称，否则它将会自动生成


**entity** 个人账号名称 或者是 team
