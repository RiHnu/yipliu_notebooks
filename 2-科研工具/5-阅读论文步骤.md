# 三遍法

## 第一遍

1. 仔细阅读标题、摘要和introduction
2. 阅读 section, sub-section 的标题
3. 看数学公式，检查下潜在的理论是否正确
4. 阅读 Conclusions
5. 看参考文献

读完后，需要理解
- 本文的 Category
- 与本文的相关文献有？
- 本文的假设是否有效？ -> 重点
- 贡献
- 是否well written

## 第三遍



# 写论文重点
- Section, sub-section 标题
- Concise and comprehensive 摘要
- 给出一个图 -》非常重要  (graphical abstract)
